Our 10-Month Curriculum

Step by step, we build coding skills and confidence.

Month-by-Month Breakdown

February: Introduction to Coding Concepts

Introduction to block-based coding using Scratch. Understanding coding logic, loops, and conditionals.

March: Storytelling Through Coding

Create interactive stories using Scratch. Learn sequencing and debugging.

April: Building Simple Games

Develop basic games like Pong using Scratch. Introduction to sprite movement and event-driven programming.

May: Introduction to Web Development

Learn HTML basics and build a simple webpage: formatting text, images, and links.

June: Styling Web Pages

Learn how to style webpages using CSS. Add fonts, colors, and layouts.

July: Animating Web Pages

Adding animations and transitions with CSS and JavaScript.

August: Game Development with JavaScript

Build interactive games using JavaScript. Explore variables, functions, and event listeners.

September: Advanced Game Logic

Develop more complex games with multiple levels and advanced JavaScript functions.
